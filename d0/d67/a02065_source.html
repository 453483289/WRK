430 /*++
431 
432 Routine Description:
433 
434  Creates a new sub-key. This is called by CmpDoCreate to create child
435  sub-keys and CmpCreateLinkNode to create root sub-keys.
436 
437 Arguments:
438 
439  Hive - supplies a pointer to the hive control structure for the hive
440 
441  ParentCell - supplies cell index of parent cell
442 
443  ParentDescriptor - Supplies security descriptor of parent key, for use
444  in inheriting ACLs.
445 
446  AccessState - Running security access state information for operation.
447 
448  Name - Supplies pointer to a UNICODE string which is the name of the
449  child to be created.
450 
451  AccessMode - Access mode of the original caller.
452 
453  Context - Supplies pointer to CM_PARSE_CONTEXT structure passed through
454  the object manager.
455 
456  BaseName - Name of object create is relative to
457 
458  KeyName - Relative name (to BaseName)
459 
460  Flags - Supplies any flags to be set in the newly created node
461 
462  KeyCell - Receives the cell index of the newly created sub-key, if any.
463 
464  Object - Receives a pointer to the created key object, if any.
465 
466 Return Value:
467 
468  STATUS_SUCCESS - sub-key successfully created. New object is returned in
469  Object, and the new cell's cell index is returned in KeyCell.
470 
471  !STATUS_SUCCESS - appropriate error message.
472 
473 --*/

476  ULONG alloc=0;

479  HCELL_INDEX ClassCell=HCELL_NIL;

483  ULONG StorageType;
