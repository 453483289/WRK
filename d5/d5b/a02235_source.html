509  // The flags values within this flag control the contents of

512  // If the context record is used as an input parameter, then
513  // for each portion of the context record controlled by a flag
514  // whose value is set, it is assumed that that portion of the

517  // portion of the threads context will be modified.
518  //
519  // If the context record is used as an IN OUT parameter to capture
520  // the context of a thread, then only those portions of the thread's
521  // context corresponding to set flags will be returned.
522  //
523  // The context record is never used as an OUT only parameter.
524  //
