322 ;
323 ; Routine Description:
324 ;
325 ; This routine is called to swap context from one thread to the next.
326 ; It swaps context, flushes the data, instruction, and translation
327 ; buffer caches, restores nonvolatile integer registers, and returns
328 ; to its caller.
329 ;
330 ; N.B. It is assumed that the caller (only callers are within this
331 ; module) saved the nonvolatile registers, ebx, esi, edi, and
332 ; ebp. This enables the caller to have more registers available.
333 ;
334 ; Arguments:
335 ;
336 ; cl - APC interrupt bypass disable (zero enable, nonzero disable).
337 ; edi - Address of previous thread.
338 ; esi - Address of next thread.
339 ; ebx - Address of PCR.
340 ;
341 ; Return value:
342 ;
343 ; al - Kernel APC pending.
344 ; ebx - Address of PCR.
345 ; esi - Address of current thread object.
346 ;
347 ;--
348 
349 ;
350 ; NOTE: The ES: override on the move to ThState is part of the
351 ; lazy-segment load system. It assures that ES has a valid
352 ; selector in it, thus preventing us from propagating a bad
353 ; ES across a context switch.
354 ;
355 ; Note that if segments, other than the standard flat segments,
356 ; with limits above 2 gig exist, neither this nor the rest of
357 ; lazy segment loads are reliable.
358 ;
359 ; Note that ThState must be set before the dispatcher lock is released
360 ; to prevent KiSetPriorityThread from seeing a stale value.
361 ;
