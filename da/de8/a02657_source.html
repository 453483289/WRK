32 Routine Description:
33 
34  This function returns the base of an image that contains the
35  specified PcValue. An image contains the PcValue if the PcValue
36  is within the ImageBase, and the ImageBase plus the size of the
37  virtual image.
38 
39 Arguments:
40

43  module contains the PcValue.
44

46  PcValue. This value must be added to any relative addresses in
47  the headers to locate portions of the image.
48 
49 Return Value:
50 
51  NULL - No image was found that contains the PcValue.
52 
53  NON-NULL - Returns the base address of the image that contain the
54  PcValue.
