523 /*++
524 
525 Routine Description:
526 
527  Initialize a hive.
528 
529 Arguments:
530 
531  CmHive - pointer to a variable to receive a pointer to the CmHive structure
532 
533  OperationType - specifies whether to create a new hive from scratch,
534  from a memory image, or by reading a file from disk.
535  [HINIT_CREATE | HINIT_MEMORY | HINIT_FILE | HINIT_MAPFILE]
536 
537  HiveFlags - HIVE_VOLATILE - Entire hive is to be volatile, regardless
538  of the types of cells allocated
539  HIVE_NO_LAZY_FLUSH - Data in this hive is never written
540  to disk except by an explicit FlushKey
541 
542  FileType - HFILE_TYPE_*, HFILE_TYPE_LOG set up for logging support
543 
544  HiveData - if present, supplies a pointer to an in memory image of
545  from which to init the hive. Only useful when OperationType
546  is set to HINIT_MEMORY.
547 
548  Primary - File handle for primary hive file (e.g. SYSTEM)
549 
550  Log - File handle for log hive file (e.g. SOFTWARE.LOG)
551 
552  External - File handle for primary hive file (e.g. BACKUP.REG)
553 
554  FileName - some path like "...\system32\config\system", which will
555  be written into the base block as an aid to debugging.
556  may be NULL.
557 
558  CheckFlags - Flags to be passed to CmCheckRegistry
559 
560  usually this is CM_CHECK_REGISTRY_CHECK_CLEAN, except for the system hive
