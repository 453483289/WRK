78

80 
81  This routine stands on its head to produce a copy of the specified
82  process's address space in the process to initialize. This
83  is done by examining each virtual address descriptor's inherit
84  attributes. If the pages described by the VAD should be inherited,
85  each PTE is examined and copied into the new address space.

88  become shared, copy-on-write, between the two processes.
89 
90

92

94  cloned.
95 
96  ProcessToInitialize - Supplies the process whose address space is to
97  be created.
98 
99 Return Value:
100 
101  None.
102 
103 Environment:
104 
105  Kernel mode, APCs disabled. No VADs exist for the new process on entry
106  and the new process is not on the working set expansion links list so
107  it cannot be trimmed (or outswapped). Hence none of the pages in the
108  new process need to be locked down.
