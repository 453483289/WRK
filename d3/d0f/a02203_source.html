281 /*++
282 
283 Routine Description:
284 
285  If necessary, poll for a request to break-in from the debugger.
286  This function should be called by routines that run at an IRQL
287  above clock level that want to be broken in by CTRL-C requests
288  from the debugger. Crashdump and hiber, for example, run at
289  HIGH_LEVEL and explicitly need to poll for breaking.
290 
291 Arguments:
292 
293  None.
294 
295 Return Value:
296 
297  None.
298 
299 --*/
300 {
301  if (KdDebuggerEnabled && KdPollBreakIn()) {
302  DbgBreakPointWithStatus (DBG_STATUS_CONTROL_C);
303  }
304 }
