181  CapturedLength. The caller should be able to assume that it points to
182  valid data that will not change. In addition, the SecurityDescriptor
183  parameter (which will receive the result of the query operation) must
184  be probed for write up to the length given in CapturedLength. The
185  security method itself must always write to the SecurityDescriptor
