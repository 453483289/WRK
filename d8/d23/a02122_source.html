71 /*++
72 
73 Routine Description:
74 
75  This function returns an LUID value that is unique since the system
76  was last rebooted. It is unique on the system it is generated on
77  only (not network wide).
78 
79  There are no restrictions on who can allocate LUIDs. The LUID space
80  is large enough that this will never present a problem. If one LUID
81  is allocated every 100ns, they will not be exhausted for roughly
82  15,000 years (100ns * 2^63).
83 
84 Arguments:
85 
86  Luid - Supplies the address of a variable that will receive the
87  new LUID.
88 
89 Return Value:
90 
91  STATUS_SUCCESS is returned if the service is successfully executed.
92 
93  STATUS_ACCESS_VIOLATION is returned if the output parameter for the
94  LUID cannot be written.
95 
96 --*/
97 
98 {
99 
100  KPROCESSOR_MODE PreviousMode;
101 
102  //
103  // Get previous processor mode and probe argument if necessary.
104  //
105 
106  try {
107  PreviousMode = KeGetPreviousMode();
108  if (PreviousMode != KernelMode) {
109  ProbeForWriteSmallStructure((PVOID)Luid, sizeof(LUID), sizeof(ULONG));
